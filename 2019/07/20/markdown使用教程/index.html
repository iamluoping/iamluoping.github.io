markdown使用教程

一级标题

`# 标题`

例：# 123

二级标题

`## 标题`

例：## 123

六级标题

`###### 标题`

例：###### 123

换行

`内容末尾加两个及以上空格`

倾斜

`*content*`

例：*123*

加粗

`**content**`

例：**123**

下划线

`***`

例：***

数字列表

`1. content`

例：1. 123

符号列表

`* content` //`*`可以用`-`代替

例：* 123 //必须在标题下一行使用

内容引用

`1左边键引用内容`

例：123

网络图片引用

`![图片名](URL地址)`

例：![123](https://avatar.csdn.net/E/0/5/3_weixin_40612082.jpg)

本地图片引用

`![图片名](本地相对地址)`

例：![123](./1.jpg)

链接

`[链接名](地址)`

例：[百度](https://www.baidu.com)
